Instructions to run the project:
1. Open the file called image_manipulation.
2. Click ‘Run’ to display the image
3. Press 1 to apply a grayscale filter
4. Press 2 to apply a high-contrast filter
5. Press 3 to apply a gaussian blur
6. Press 4 to apply an edge-detection filter
7. Press 5 to apply a high saturation filter
8. Press 0 to revert back to the original image
